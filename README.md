# Gospel dashboard (client)

> Gospel用户控制台

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
## develop settings
``` JavaScript
git config commit.template git_commit.template
```
